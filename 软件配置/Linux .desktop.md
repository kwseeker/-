# Linux 桌面入口文件（.desktop）

位置： `/usr/share/applications/*.desktop`

## GoLand

```properties
[Desktop Entry]
Version=1.0
Type=Application
Name=GoLand
Icon=/opt/goland/bin/goland.svg
Exec="/opt/goland/bin/goland.sh" %f
Comment=Go IDE
Categories=Development;IDE;
Terminal=false
StartupWMClass=jetbrains-goland
```
